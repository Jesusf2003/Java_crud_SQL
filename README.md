# CRUD con Java y SQL

En este repositorio se realizarán aplicaciones Java implementando
una base de datos relacional y no-relacionales.

## Herramientas y tecnologías

- Eclipse IDE Enterprise 2022-03
- Java 17
- Apache Maven 3.8.1

### Se hará uso de:

- Java Swing / AWT
- Java JPA Desktop
- JavaFX

### Y para la base de datos se hará uso de:

- Relacional: PostgreSQL
- No - Relacional: MongoDB
